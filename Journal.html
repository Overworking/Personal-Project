Day 16: Preparing to make a website

Download an IDLE(program to run the code) called atom. (I used brackets after Day 20.)
Since making a website from the very base is such time-consuming, I used the clean blog template as a starter.
Installed initial packages to write and perform the code easily. The packages I used are the followings:
- atom-beutify: Formats and organizes the code
- atom-html-preview: Add a website preview function to the program
- emmet: Adds code expansion
- atom-ternjs: Adds code intelligence
- pigments: Displays the colour in Atom
